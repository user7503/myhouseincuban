# MyHouseinCuban 🏠

Plataforma profesional de venta de propiedades en Cuba, con diseño inspirado en Airbnb.

## Características

- Diseño moderno y responsive
- Búsqueda avanzada con filtros
- Galería de imágenes con carrusel
- Mensajería entre compradores y vendedores
- Panel de vendedor completo
- Panel de administrador con gráficos
- Gestión de usuarios y propiedades

## Instalación

1. Clona o descarga el proyecto.
2. Crea un entorno virtual (opcional).
3. Instala dependencias: `pip install -r requirements.txt`
4. Ejecuta: `python app.py`

## Acceso

- Admin: admin@myhouseincuban.com / admin123
- Vendedor: seller@myhouseincuban.com / seller123

## Producción

Para entornos de producción, configura las variables de entorno adecuadas y usa un servidor como Gunicorn.
